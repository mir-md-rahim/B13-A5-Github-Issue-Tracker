- 1️⃣ What is the difference between var, let, and const?
  Answer: var, let and const is type of variable declaration in javascript. difference:

1.  var: This is old type variable declaration.
2.  let: This type variable's value can be changed next time.
3.  const: This type variable's value can not be changed next time.

- 2️⃣ What is the spread operator (...)?
  Answer: Expands an array or object into separate elements.

- 3️⃣ What is the difference between map(), filter(), and forEach()?
  Answer: Difference:

1.  map() transforms each element and return a new array.
2.  filter() test a conditon and return a new array with only the element that pass test.
3.  forEach() executes a function on each element of an array, but does not return anything.
